# Tripfolio (Prototype)

Tripfolio is a prototype travel portfolio web application.  
This project is intended as a **reference prototype** showcasing layout, styling, and UI interactions — not a production-ready app.

---

## ✨ Features

- **Responsive App Layout**  
  Flexible sidebar + main content design with smooth transitions.

- **Collapsible Sidebar**  
  Expand/collapse functionality with animated logo and navigation links.

- **Mobile-Friendly Navigation**  
  Sidebar adapts to mobile screens with overlay support.

- **Page Sections (SPA-like)**  
  Each section can be toggled active/inactive for single-page navigation.

- **Interactive UI Elements**  
  - Circular progress ring with smooth animation  
  - Custom modal overlay with scale transitions  
  - Toast notifications with slide-in effect  
  - Loading overlay with animated spinner

- **Dark Mode Support**  
  Styles adapt for light and dark themes.

- **Professional UI Tweaks**  
  Hover states, active states, and icon sizing for a polished look.

---

## 📂 Project Structure

- `style.css` → Core stylesheet containing layout, sidebar, modal, toast, loading overlay, and button styles.
- (Other files like `tripfolio.html` or `app.js` can be added depending on your prototype setup.)

---

## 🚀 Getting Started

1. Clone or download this repository.
2. Open `index.html` in your browser (ensure it links to `style.css`).
3. Explore the sidebar, modal, toast, and other UI components.
